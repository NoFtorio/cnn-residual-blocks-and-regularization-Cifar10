# Comparación de Arquitecturas Convolucionales en CIFAR-10

## Descripción del Proyecto
Este proyecto analiza el impacto de las conexiones residuales y la normalización por lotes (Batch Normalization) en el entrenamiento de redes neuronales profundas utilizando el dataset CIFAR-10 en PyTorch.

## Arquitecturas Evaluadas
- **Modelo A**: Red Convolucional Estándar (Sin conexiones residuales).
- **Modelo B**: Red Convolucional con Bloques Residuales (ResNet).
- **Modelo C**: Red Convolucional Residual con Batch Normalization.

## Requisitos
- Python 3.x
- PyTorch
- Torchvision
- Pandas / Matplotlib / Seaborn / tqdm
